# ArtWalk Deep Ellum Archive

## Overview

ArtWalk Deep Ellum is a digital archival project dedicated to documenting and preserving the history of ArtWalk Deep Ellum (1985–1997).

This repository serves as an evolving public record built from photographs, flyers, documents, personal accounts, and community contributions.

## Purpose

The goal of this project is to collect, organize, and preserve historical materials related to ArtWalk Deep Ellum in order to maintain an accessible cultural record for future reference and research.

## Scope

This archive focuses on:

- Historical photographs  
- Event flyers and printed materials  
- Personal memories and written accounts  
- Media and documentation related to ArtWalk Deep Ellum  
- Community-submitted archival content  

## Status

This project is currently under development. Structure, materials, and documentation are actively being organized and expanded.

## Contributing

Contributions of historical materials, stories, or references are welcome as this archive develops. More formal submission guidelines will be added as the project grows.
